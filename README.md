# SecondBrainAI
SecondBrainAI is a fully local knowledge management system powered by open-source LLMs. It allows users to upload and process documents (PDF, text, web pages, audio), automatically extract insights, build semantic search indexes, and interact with their personal knowledge base via an AI assistant.

The system is designed as a “second brain” that not only stores information but also understands relationships between concepts and helps users learn faster and more effectively.

All processing runs locally — no external AI APIs are required.

⸻

Key features:

* Local-first AI assistant (no cloud APIs)
* Document ingestion (PDF, TXT, Markdown, web pages, audio)
* Semantic search over personal knowledge base
* Automatic summarization and key idea extraction
* AI chat interface over your documents (RAG-based)
* Modular architecture for future knowledge graph expansion
